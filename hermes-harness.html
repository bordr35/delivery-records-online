How It Works

This is the mental model: pick a scenario, read the prompt, inspect the data payload, then run it against Hermes.

1. Pick a scenario

Each scenario is a small test. One checks summaries, another checks merchant friction, and so on.

2. Read the prompt

The prompt is the actual instruction Hermes would receive. That is the behavior you are testing.

3. Run or copy

Either send the payload to your endpoint or copy the prompt and paste it into another agent tool.
